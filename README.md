# Enlightened Bharat School Website

A modern school website built using React, Vite, Firebase Firestore, and Firebase Storage.

## Features

- School information and admission portal
- Online admission form
- Firebase Firestore integration
- Admin dashboard for managing admissions
- Delete admission records
- Gallery image upload and management
- Search, sorting, and pagination
- CSV export functionality
- View details modal
- Custom delete confirmation modal
- Toast notifications
- Responsive UI design

## Tech Stack

- React.js
- Vite
- Firebase Firestore
- Firebase Storage
- React Router
- React Toastify
- CSS3
